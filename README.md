#backup skin tool
<h2>Usage</h2>
<pre>  python3 ./backup_skin.py</pre>
<dl>
    <h2>Options</h2>
    <dt>Skin folder:</dt>
    <dd>
        Base skin folder, this folder should contain
        <strong>Art</strong>, <strong>UI</strong> and other needed folders
        for mod skin.
    </dd><br><br>
    <dt>Output Folder Name:</dt>
    <dd>
        Folder that contain the backup for skin
    </dd>
</dl>
